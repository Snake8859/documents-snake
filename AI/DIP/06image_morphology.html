# 图像形态学处理

数学形态学：分析几何形状和结构的数学方法，建立在集合代数的基础上，用集合论方法定量描述集合结构的学科。1985年以后成为分析图像几何特征的工具。

数学形态学图像处理的基本思想：使用具有一定形态的结构元素，去度量和提取图像中的对应形状，如边界、骨架、凸包等，以达到对图像进行分析和识别的目的。

数学形态学图像处理意义：可以简化图像数据，保持它们基本的形状特性，并除去不相干的结构。

数学形态学的基本运算：膨胀、腐蚀、开运算和闭闭运算,击中击不中变换。

# 腐蚀和膨胀

# 膨胀（使图像扩大）

解释：A 被 B 膨胀是所有位移 x 的集合， B 的映射与A至少有一个元素是重叠的。换言之，用 B 膨胀 A 得到的集合是B 的映射的位移与 A 至少有一个非零元素相交时 B 的原点 x 位置的集合。

等价表达：

膨胀的算法：

  1. 用反射结构元素，扫描图像的每一个像素；
  2. 用反射结构元素与其覆盖的二值图像做或运算
  3. 如果都为0，结果图像的该像素为0，否则为1

等价算法：用结构元做成0，1构成的模板，与图像做卷积，所有大于0的地方就是膨胀结果。

膨胀的作用：

  1. 用3x3的结构元时，物体的边界沿周边增加一个像素
  2. 把目标周围的背景点合并到目标中，目标之间存在细小的缝隙，膨胀可能将不同目标连通在一起
  3. 填补分割后物体中的空洞

案例理解：

基于膨胀定义本身的运算：反射结构B，遍历图像A的每个像元，若B的映射与A至少有一个元素是重叠的，则原点位置像素变为1，否则是0。

基于位移运算的膨胀运算：结构B当成三个方向对图像A：自身，向右，向左，再将三个图像A结果求并集。

# 腐蚀（使图像缩小）

解释：A 被 B 腐蚀是所有位移 x 的集合， 其中 B 平移 x 后仍包含于 A 中。换言之，用 B 腐蚀 A 得到的集合是B 完全包含在 A 中时 B 的原点位置的集合。

等价表达：

腐蚀的算法：

  1. 用结构元素，扫描图像的每一个像素；
  2. 用结构元素与其覆盖的二值图像做与（且）运算
  3. 如果都为0，结果图像的该像素为0，否则为1

等价算法：用反射结构元做成0,1构成的模板，与图像做卷积，结果等于模板和的值的点就是腐蚀的结果。

腐蚀的作用：

  1. 用3x3的结构元时，物体的边界沿周边减少一个像素
  2. 消除掉图像中小于结构元大小的目标物体
  3. 若物体之间有细小的连通，选择适当的结构元，可以将物体分开
  4. 不同的结构元及其不同的原点，产生不同的结果

案例理解：

基于腐蚀定义本身的运算：结构B，遍历图像A的每一像元；若B包含在A中，则原点像素是1，否则为0。

基于位移运算的腐蚀运行：结构B进行映射，将映射结构B当成三个方向对A：自身，向左，向上平移，再将三个图像A结果求交集。

# 开运算与闭运算

# 开运算：先腐蚀再膨胀

使用结构元素 B 对集合 A 的开运算是用 B 对 A腐蚀，然后用 B 对腐蚀结果进行膨胀。

# 闭运算：先膨胀再腐蚀

使用结构元素 B 对集合 A 的闭运算是用 B 对 A膨胀，然后用 B 对腐蚀结果进行腐蚀。

# 开闭运算的几何解释

开运算相当于，一个腐蚀结构在锐角三角形内滚了一圈，然后再用膨胀结构在上一次结构上再滚一圈，这样子尖锐的部分将得到平滑，断开狭窄的间断喝消除细的突出物。

闭运算相当于，一个膨胀算子在尖锐凹槽内滚过去，然后再用腐蚀结构在上一次结构再滚一次，这样子尖锐的部分也得到平滑，消除狭窄的简短喝长细的鸿沟，消除小的孔洞并填充轮廓线中的断裂。

  • 腐蚀

    • 目标收缩

    • 断开狭窄的连接

  • 膨胀

    • 使目标扩张
    • 平滑物体边界
    • 连接狭窄的断开

    • 与腐蚀相似但是比腐蚀较少破坏性

  • 闭运算

    • 与膨胀相似但是比膨胀较少破坏性

  • 细化

    在不破坏连通性的前提下，细化图像

    算法实现：

    1. 做腐蚀操作，但不立刻删除像素，只打标记

  • 基于距离变换的骨架提取

    距离变换图算法是一种针对栅格图像的特殊变换，是把二值图像变换为灰度图像，其中每个像素的灰度值等于它到栅格地图上相邻物体的最近距离。

    基本思想 ：是把离散分布在空间中的目标根据一定的距离定义方式生成距离图, 其中每一点的距离值是到所有空间目标距离中最小的一个。对于距离的量度是通过四方向距离（又称“城市块距离”或“出租车距离”）的运算来实现的，即只允许沿四个主方向而不允许沿对角方向进行跨栅格的最小路段的计数。因此，每个路段为一个像元边长。

    串行距离变换算法：

    从距离变换图中提取出具有相对最大灰度值的那些像元所组成的图像就是图像的骨架。

    算法过程：

    1. 求取二值图像的边界
    2. 对边界二值图像求取距离变换
    3. 求距离变换图中的局部极大值
    4. 落入原二值图像中的局部极大值就是图像的骨架

# 裁剪

裁剪方法实际上是对细化和骨架绘制算法的补充，因为要清除这些算法产生的一些不必要的附加成分。

其主要过程为：

  1. 用一系列被设计用来检测终点的结构元素对A进行三次细化

  2. 得到建立在X1中,包含的所有端点的集合X2

  3. 对端点进行三次膨胀处理，用集合A作为消减因子；

  4. 结果为X3和X1的并集
