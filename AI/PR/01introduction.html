# 人工智能

人工智能=人工+智能

人工智能是关于知识的学科，怎样表示、获取和应用知识；

核心目标：让机器去完成只有人能够完成的智能工作。

人工智能所包含的体系：

  • 模式识别
  • 机器学习
  • 数据挖掘
  • 计算机视觉
  • 自然语言处理

# 模式识别

学术描述1：模式识别是人工智能的一个分支，其基本任务是从标记样本中训练识别系统或者从无标识样本中发现模式。

学术描述2：模式是指存在于时间和空间中可观测性、可度量性和可区分性的信息；模式识别是对模式进行分析与处理， 进而实现描述、辨识、分类与解译。

其他角度理解：

  • 人和动物通过感知从周围环境获取信息。感知就是模式识别过程。
  • 使计算机模仿人的感知能力，从感知数据中提取信息（判别物体和行为、现象）的过程。

特性：

  • 可观察性
  • 可区分性
  • 相似性

基本概念理解

模式识别：确定一个样本的类别属性的过程，即把某一个样本归为多个类型中的某一个。

模式：对研究对象（客体)的一种抽象化描述，是对客体的统称。【简单理解：每一个类组成模式】

样本：一个具体的研究对象。一个样本必须量化或者结构化才能被计算机识别，这就是模式的特征。

特征：能描述模式特征的量。

模式分类是模式识别的核心研究内容，相关问题包括模式描述、特征提取、特征选择、聚类、分类器设计等。

模式分类取决于具体的数据对象，模式识别的研究内容还包括信号 /图像/视频理解、视觉目标分类、图像/视频检索、文本分类等，以及面向应用的技术研究。

  • 如何特征提取（量化）？
  • 如何特征选择？
  • 如何选择分类器？
  • 分类器训练过程与测试？

# 机器学习

机器学习：专门研究计算机怎样模拟或实现人类的学习行为，以 获取新的知识或技能，并重新组织已有的知识结构使之不断改善自身的性能。

解决的问题：

  • 如何在经验学习中改善算法性能
  • 用数据或以往的经验，优化计算机程序的性能标准

# 三者区别

机器学习，数据挖掘和模式识别三者区别，关系和理解？

  • 机器学习侧重于学习模型的构建，强调从数据中学习，并最大化某个学习目标
  • 数据挖掘侧重于从无监督数据中发现未知的知识，强调其商业应用
  • 模式识别侧重于强调描述，解释和可视化一个特定的模式

# 模式描述

模式描述是将模式转化为向量表示的过程，即特征描述与特征提取。

模式的两个层次：

  • 样本（sample，instance，example）
  • 类别（class,category)

例如：100个样本，10个类别

识别对象表示：特征

# 模式分类

模式识别核心技术为模式分类，即采用合适的分类器对模式进行分类。

分类器表示：

  • 判别模型
  • 生成模型

学习方法分类：

  • 监督学习：训练样本有类别标签
  • 无监督学习：训练样本无标签，主要得到数据结构表示或分布
  • 半监督学习：训练样本一部分有类别，一部分没有
  • 增强学习：学习过程中给出奖惩信号
  • 自适应学习：测试样本分布发生变化，分类器参数自适应
