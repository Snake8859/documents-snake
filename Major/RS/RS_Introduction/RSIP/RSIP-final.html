# 遥感数字图像信息提取及制图表达

# 感兴趣目标与对象提取

遥感图像中的感兴趣目标是指图像中用户最为关注的目标地物。

感兴趣目标提取原因主要是由于遥感图像庞大的数据量，后续处理及分析工作量大。

感兴趣目标提取的作用

  • 去除冗余数据、提高图像处理和分析速度、排除其它无关数据的干扰
  • 获得高分辨率遥感图像上的目标对象

# 图像分割

# 阈值分割法

阈值法是利用图像中目标物体和背景在灰度分布上的差异，通过设定阈值，判断图像像元是属于目标区域还是背景区域，从而实现图像分割，生成二值图像。

阈值法的核心就是阈值的最优选取，我们既可以根据整幅图像的全部信息进行阈值选取，也可以根据其局部信息进行阈值选取。

对于灰度直方图具有明显峰谷的图像，阈值可以直接通过查看图像的灰度直方图获得。

当图像的灰度直方图不具备峰谷特征，或者当我们想要通过图像自动获取分割阈值时，就需要采用一些方法来自动搜索最佳阈值，这些搜索最佳阈值方法主要有均匀性度量法、最大类间距离法、最大类间方差法、最大熵法等。

# 边界分割

图像边缘是指图像局部区域亮度变化显著的部分，该区域的灰度剖面一般可以看作是一个阶跃。图像边缘是物体的轮廓或物体不同表面之间的交界在图像上的反映，是图像局部特性不连续（或突变）的结果。

边界是具有相同灰度值的最大连通域的外廓线，边界是孤立且闭合的，它由边缘连接而成。

边界分割方法是利用不同区域间像元灰度不连续的特点检测出区域间的边缘，从而实现图像分割。

边缘检测只是所有基于边界分割方法的第一步，当完成边缘检测之后，还需将相互不连通的边缘连接起来构成封闭的区域才能实现目标与背景的分离，换句话说，边界分割法就是沿着目标周边闭合的边界线将其包围的区域剪切出来。

边缘检测的方法主要有基于微分算子的边缘检测、综合边缘检测和形态学梯度的边缘检测。

  • 基于微分算子的边缘检测

    原理：将微分算子作用于原图像，计算出各像元的梯度值。由于目标或背景内部像元的梯度值较低，而其边缘的梯度值较高，因此可以通过对梯度图像的灰度值进行统计来选择合适的阈值对梯度图像进行分割以得到边缘。

# 区域提取分割法

  • 区域生长法

    区域生长法是利用相邻像元之间的相似性将像元扩充到一个区域。首先在待分割区域选取一小块作为种子区域，然后将与其性质相同或相似的邻域像元添加至区域并作为新的种子，再以新的种子向外扩张直到无法继续归并像元，也就达到终止状态。区域生长法是一种迭代方法。由此可知，区域生长的好坏取决于种子点的选取、生长准则和终止条件。

    • 种子点的选取

      种子点的选取：一般需借助先验知识进行选取。要是对具体问题没有先验知识，常可借助生长所用准则对图像进行聚类分析，接近聚类中心的像元可作为种子像元。

    • 生长准则

      生长准则：即在生长过程中归并邻域像元的相似性准则，相似性准则可以是灰度级、颜色、组织、梯度或其它特性。生长准则的选取不仅依赖于具体问题本身，也和所用图像数据的种类有关。另外还需考虑像元间的连通性和邻近性，否则有时的分割结果没有意义。常用的生长准则有：基于区域灰度差、基于区域内灰度分布统计性质和基于区域形状。

    • 终止条件

      终止条件：决定区域生长的停止条件或规则，一般情况下，无法找到满足生长准则的像元时就停止。但常用的基于灰度、纹理、颜色的准则大多只考虑图像的局部性质，并没有充分考虑生长的过程，需要设置终止条件从全局进行把关。

    相似性准则是种子像元与邻接灰度像元的灰度差不大于2，终止条件与相似性准则一样

    相似性既可以是两个空间邻接像元之间的最小灰度差，也可以是种子像元集合的平均灰度值与邻域像元间的最小灰度差，但它们产生的分割结果会有差异：当图像中存在灰度缓慢变化的区域时，如果相似性采用两个空间邻接像元之间的最小灰度差来度量则有可能会将不同区域逐步合并而产生错误的分割结果；为了克服这个问题，可以不用新像元的灰度值去与邻域像元的灰度值进行比较，而是用新像元所在区域的平均灰度值去与各邻域像元的灰度值进行比较。

  • 区域分裂合并法

    区域分裂合并法的设计思想是：如果一个大区域的属性不一致，则将区域不断分裂为若干不相交的子区域，直到每个子区域内部属性都一致；分裂之后，如果相邻的子区域具有相同的性质，则将相邻区域进行合并。区域分裂合并法的关键是确定分裂准则和合并准则，也就是一致性准则，通常以区域的某种特征（如灰度、颜色、统计特征等）均匀性作为准则。

    分裂原则为同一区域灰度方差不大于1，合并原则为相邻区域灰度差不大于2

# 形态学分水岭分割

分水岭分割是将灰度图像看作地形曲面，灰度值表示高程，每一个局部极小值及其影响区域称为集水盆，而集水盆的边界则形成分水岭。

分水岭的概念和形成可以通过模拟浸水过程来说明：在每一个局部极小值表面，刺穿一个小孔，然后把整个模型慢慢浸入水中，随着浸入的加深，每一个局部极小值的影响域慢慢向外扩展，在两个集水盆汇合处构筑大坝，即形成分水岭，根据分水岭可以将图像分割成不同的区域。

形态学分水岭分割的最大优点是能直接得到封闭的边界，其缺点是最容易产生过度分割（即一个实际的目标被分割成了多个目标）

过度分割问题主要是由图像中存在的噪声或者局部不规则现象所导致，目前解决过度分割有两类常用的方法：

  • 对图像进行预处理，如进行平滑、梯度变换、标记等
  • 分割后的图像进行再处理，如区域合并

# 二值图像处理

# 四近邻与八近邻

四近邻是指当前像元最近的四个像元，即上下左右4个像元，不包含对角像元。而八近邻则是指当前像元周边的八个像元，包括上下左右4个像元和对角4个像元。

# 四连通与八连通

四连通是指当前目标像元在其四近邻中存在同类像元；同样，八连通是指当前目标像元在其八近邻中存在同类像元 。所以，四连通成立的时候，八连通一定成立；但八连通成立，四连通不一定成立。

# 内部点与边界点

在每个连通域中，与背景相邻接的点称为边界点，与背景不邻接的点称为内部点。在四连通定义下，如果当前目标像元的八近邻像元中没有背景像元，则该像元为内部点；反之，为边界点。在八连通定义下，如果当前目标像元的四近邻像元中没有背景像元，则该像元为内部点；反之，为边界点。

# 空洞点填补和碎块消除

空洞点填补通常可以采用数学形态学的闭运算实现。碎块消除通常可以采用数学形态学的开运算实现。 通常为了同时进行空洞点填补和碎块消除，开运算和闭运算都是以组合的方式出现，所以可以对图像进行闭-开运算或开-闭运算。

# 对象提取

遥感图像中的对象：即遥感图像上具有相同特征（如光谱、纹理和空间组合关系等特征）的“同质均一”单元，“同质均一”不仅体现在光谱域上，也体现在空间域上。

二值图像存在多个连通域时，需将各个连通域分开来单独分析其属性，因此需对各目标单元进行识别并赋以单独的编号（即贴标签）。同时，为了方便对对象的形态特征进行分析，还需将各目标单元进行矢量化，以提取各目标单元的封闭边界轮廓。

# 贴标签

贴标签处理是对二值图像的每个不同连通域进行不同的编号以区分不同的连通域。

四连通和八连通意义下的连通域存在差别，所以在贴标签时需区别对待。

贴标签涉及两个关键步骤：首先是遍历图像贴新的标签；其次是对已贴好的标签进行校正。

# 边界提取

目的：栅格数据较适合对各目标物的统计特征进行描述，但不适合描述其形态特征；而矢量数据适合描述各目标物的形态特征。为此，需要找出各目标物的矢量边界。

方法：边界提取通常是对贴标签的结果图像进行递归搜索运算以获得各目标物的边界坐标，将这些边界坐标首尾连接起来则得到了闭合的矢量边界。

# 特征提取与选择

特征的理解：在两个或多个目标物之间具有差异的属性就是物体的特征（特征属性或特征变量），也就是可以被用来区分目标物的属性。特征具有相对性，它永远是相对于两个或多个相互比较的目标物而言

特征提取和选择的目的：区分不同的目标物。就遥感数据处理而言，就是服务于遥感图像分类。

特征的类型：

  • 原始特征：能直接反映物体之间差异的原始属性。
  • 衍生特征：把某些或者所有原始属性通过变换生成新的特征变量，从而增强目标物之间的可分性，这种通过变换方式得到新特征变量的过程就是特征提取。

遥感影像的特征类型：

  • 原始特征：光谱信息
  • 衍生特征：全局性的光谱特征统计变量和局部性的空间特征（特征提取包括光谱特征提取和空间特征提取）

# 特征提取

# 光谱特征提取

光谱特征提取的基本思想就是对多种属性进行某种线性或非线性组合得到综合指标。

目的：在多光谱（尤其是高光谱）图像中，邻近波段之间往往具有高度的相关性，存在着大量冗余和重复的信息，需从这些数据中提取那些无冗余的有效信息来识别目标地物。

方法：主成分变换、最小噪声分离、缨帽变换、植被指数等。

  • 光谱特征提取：主成分变换

    通过对图像上的每个像元的光谱特征向量进行线性变换，从而产生一幅新的图像，新图像的各分量（小于或者等于输入图像的波段数）之间差异最大、但相关性最小（即相互正交），各分量按方差大小排序。

    目标：多光谱遥感影像在经过主成分变换后，前面几个主成分包含了原始图像90%以上的信息量，因此一般会选择前几个分量作为备选的分类特征变量。

  • 光谱特征提取：最小噪声分离变换

    最小噪声分离变换通过对信号与噪声的分离，所获得的分量是按信噪比由高到低排序的。

    目标：与主成分变换类似，最小噪声分离变换之后也是通常选择前几个信噪比高的分量作为备选的分类特征变量

  • 光谱特征提取：缨帽变换

    缨帽变换是一种特殊的主成份分析。缨帽变换的变换系数是固定的，其旋转后坐标轴不是指向主成份方向，而是指向与地物有密切关系的方向，特别是与植物生长过程和土壤有关的信息，通常变换后的可用分量包括亮度指数、绿度分量、黄度分量和湿度分量，同一传感器不同图像产生的分量之间可进行相互比较。

    目标：利用缨帽变换进行特征提取时，我们可以根据实际需求选择不同的分量进行分类，如植被分类可以考虑选择绿度分量，土壤水分遥感反演可以考虑选择湿度分量。

  • 光谱特征提取：植被指数

    由于绿色植物对红光波段具有强吸收作用，而对近红外波段具有高反射作用，因此绿色植物对这两个波段的光谱响应具有明显的反差，植被指数正是通过对这两个波段的组合来进一步增强植被信息，并且使新的变量对植被长势、生物量等具有一定指示意义。

# 空间特征提取

空间特征属于局部统计变量，反映图像局部的灰度变化、像元组合及其与周边的关系。

类型：纹理特征、形状特征、空间关系特征

注意：空间特征与对象尺度的关系（当图像分割尺度过细时，形状和空间关系特征无法表达）

  • 空间特征提取：纹理特征

    纹理反映了图像灰度模式的空间分布，包含了图像的表面信息及其与周围环境的关系，更好地兼顾了图像的宏观结构与微观结构。纹理就是由纹理基元按某种确定性的规律或者某种统计规律排列组成的，纹理具有局部的随机性和整体的统计规律性。

    表现：

    1. 某种局部的序列性在比该序列更大的区域内不断重复出现
    2. 序列是由基本部分（即纹理基元）非随机排列组成的
    3. 在纹理区域内各部分具有大致相同的结构

    遥感图像纹理特征常用提取方法：统计法（灰度共生矩阵）

    • 灰度共生矩阵

      本身不能作为纹理特征，而是通过灰度共生矩阵的能量或者相关统计特征来表征图像的纹理特性。

      灰度共生矩阵构建的基本思路：

      假定在图像中开一个固定大小的窗口，并将该窗口在图像的某一方向上移动一段距离得到一个新的窗口，两个窗口相应位置的像元则构成了像元对，像元对中的两个像元的灰度值则构成了一组灰度值对组合，如果图像具有一定的纹理特征，则每个像元对的灰度值对组合不是唯一的，也就是存在重复出现的可能，那么这个区域内的灰度值对存在的统计规律在一定程度上就可以反映这个区域的纹理特性。

      灰度共生矩阵的数学算法：

      假如在窗口中任意取出一像元，设定其坐标为(x, y)、灰度值为i，当窗口在θ方向偏移距离d时，对应的像元坐标位置为(x+a, y+b)、灰度值为j（其中a=dcosθ，b=dsinθ），这两个像元构成的灰度值对组合为 (i, j)。统计出原窗口与新窗口中每个灰度值对组合出现的次数并排成一个矩阵，进而生成一个表示k个像元对中灰度值对组合(i, j)出现频率的概率矩阵，即灰度共生矩阵。

      灰度共生矩阵的影响因子：

      • 偏移方向θ
      • 偏移距离d
      • 窗口大小
      • 灰度级数

# 特征选择

用于遥感图像分类的属性特征可能非常多，如果不加选择地将所有属性都作为分类特征，那么分析特征、训练模型所需的时间就会很长，模型也会变得复杂，其推广能力（泛化能力）就会下降；此外，特征变量过多会引起“维数灾难”，使得识别精度不增反降。

特征选择的标准：

  • 可区分性
  • 可靠性
  • 独立性
  • 数量少

# 技术流程

  1. 子集产生
  2. 子集评价（非监督选择和监督选择）
  3. 评价终止
  4. 结果验证

# 属性评价准则

根据与分类算法的关联程度，属性评价准则大体上可以分成两类：关联准则和独立准则。

  • 关联准则

    关联准则依赖于分类算法，它是以分类算法的性能作为评价准则。

    • 监督特征选择，在特定的分类器下常采用分类准确率作为评价准则。
    • 非监督选择，在特定的聚类算法下常采用属性子集的聚类质量来作为评价准则，常用的聚类质量评价参数有类别的紧凑性、类内类间的距离和最大可能性等。

  • 独立准则

    独立准则是通过训练样本的内在特征来对所选择的属性子集进行评价，不依赖于特定的分类算法。

    • 距离度量（监督选择）

      常采用类别可分性距离、散度等指标进行评价，其基本思想就是选择对训练样本可区分性大的属性，比如对于二分类问题和两个特征X与Y，如果特征X更容易区分两个类别，则优先选择X属性作为区分特征。

      常用指标：如转换离散度和Jeffries-Matusita距离等。

    • 相关性度量

      • 监督选择：主要是评价属性和类别的相关性

        标准：如果属性X和类别A的相关性大于属性Y和类别A的相关性，则选择属性X作为区分特征。

      • 非监督选择：主要考虑属性间的相关性

        标准：属性间相关性越高，说明两个属性间的信息重叠度越高，因此在特征选择时可将相关性大的两个属性融合或者只取其中一个。

    • 信息度量（非监督选择）

      标准：主要是根据属性的信息增益进行判断，如果属性X的信息增益大于属性Y，则选择属性X作为分类特征。常采用与熵相关的各种指标来度量信息量。

# 基于先验知识的特征选择

前人的工作基础和研究经验可以给我们提供特征选择的思路。

基本思想：如果我们对研究区地物及其属性比较熟悉，已经知道某些属性可以很好地区分待分类的地物，此时我们可以根据这些先验知识直接选择这些可以区分特定地物的属性，该方法非常适合光谱信息相对较少的多光谱数据。

局限性：先验知识往往受限于所识别的地物类别及其所处的环境

# 特征组合

特征组合涉及两方面的内容：

  • 各特征参与分类的先后顺序
  • 各特征参与分类时的权重

作用：对于某些分类器来说，给各特征变量赋予不同的权重后再进行分类，则会得到不同的分类结果

决策树分类对于特征组合的体现比较典型，决策树的构建过程就体现了各特征变量出现的先后顺序；而且，某一特征可能被多次使用，也就是说该特征在分类过程中的贡献不只一次，即它的权重相较于其他特征来说要更大一些。

# 遥感图像分类

遥感图像分类是指根据感兴趣目标在遥感图像上的特征差异，判断并识别其类别属性和空间分布特征等信息的过程。分类的本质就是从观测样本的属性中选取对判断目标具有差异的特征，并利用这些特征建立判断标准，从而将观测样本划分为不同的类别。

# 遥感图像分类原理

  • 概念理解：地物在遥感影像上的每个属性均可被当作一个变量，参与分类的这些属性（也称为特征变量）构成了一个n维的特征空间。理想情况下，同类地物应该具有相同或相似的特征描述，因此他们的像元在n维特征空间中聚集在一起；而不同地物应该具有不同的特征描述，因此不同地物的像元在n维特征空间中呈现分离状态。
  • 图形理解：一个像元可以看成由n个特征组成的n维空间的一个点。同类地物的像元形成n维空间的一个点群，差异明显的不同地物会构成n维空间的若干个点群。计算机分类就是要分析特征空间这些点群的特点，如点群的位置、分布中心，从而确定点群的界限，最终完成分类任务。

# 遥感图像分类过程

# 1. 分类目的及研究区背景了解

先要对分类的目的和地区背景有所了解，比如针对分类目标，可以有简单分类或者详细分类，例如某块地区分植被和非植被的简单分类，又或者某块地区分针叶林，阔叶林等的详细分类，除此之外，还有研究地区背景了解，例如对于南方来说植被种类比较多，分布比较离散，判别比较困难，对于北方来说植被种类比较单一，分布集中，判别比较容易。

# 2. 数据获取

对研究区域和目标物所有用数据获取，看能获取哪些有用数据可以帮助分类，要注意数据空间，时间和光谱分辨率。

# 3. 数据预处理

包括辐射校正，几何校正等，需要根据需求判断是否需要进行数据预处理。

# 4. 分类类别确定和解译标志建立

# 5. 训练样本选取和评价

  • 训练样本的来源

    • 野外调查获得（实地样本形状的边长应至少大于分类影像的空间分辨率的3倍长度）
    • 在遥感影像上选取（主要参考历史图件和高分辨率遥感图像）

  • 训练样本的个数要求

    样本个数与所采用的分类方法、特征空间维数、各类的大小与分布有关。一般情况下，每一类别的训练样本个数至少是特征空间维数的10~30倍，当研究区域地物各类别的内部差异较小的时候，可以适当减少训练样本个数。

  • 训练样本的分布

    对绝大多数分类器来说，训练样本应该能够代表研究区内各分类类别的特征差异，同一类别的训练样本必须纯净。

    对于支持向量机分类器来说，使用分布在类别边缘的混合像元训练样本可以获得比使用纯净训练样本更高的分类精度。

    策略：小区域连续采集、全局均匀分布的采集策略、分层采集、地理分层采集。

  • 训练样本评价

    训练样本的评价：判断训练样本是否能表现不同类别的光谱特征，即不同类别的光谱特征向量的分离程度。

    策略：分离程度较小，应该重新选择训练样本；混分现象严重时，应该考虑把两个类别合并

    评价方法：

    • 图表法：均值图法、直方图法、特征空间多维图法

      • 均值图法

        把各类别在不同特征空间的样本点特征值求均值和标准差，构建一个折线图来判断不同类别在不同特征空间的分布情况。

      • 直方图法

        显示训练样本在某一光谱波段上的灰度频数分布。

      • 特征空间多维图法

        把训练样本的光谱特征向量显示在多维的窗口中，可以直观地展现所有样本的光谱特征分布状况。

# 7. 分类方法的选择

  • 监督分类器

    • 最大似然法

      基本思想：首先定义一个从属于某种类别的概率分布集群，然后把待分类像元落入各类别的条件概率作为判别函数，将像元落入某类别的条件概率最大的类定义为该像元的类别。

      优点：该方法考虑了各类别在不同特征变量上的内部方差以及相关性，有效地解决了训练样本重叠区的类别归属问题。

      局限性：如果两个以上的特征变量之间具有强相关性，那么协方差矩阵的逆矩阵可能不存在，所以在选择特征变量时应该注意该问题，最好选择相关性低的特征变量组合来参与分类。

    • 支持向量机分类算法

      基本思想：待分类数据点是低维空间中的点，我们希望通过升到n维以找到一个n-1维的最大间隔超平面将不同的类别区分开。

    • 反向传播（BP）神经元网络模型

      源于对生物大脑机理的仿生学研究。BP算法通过样本和标签值对神经元网络进行训练，采用反向传播算法来逐层调整各神经元的权重及截距，使得最终的输出和标签值接近。

    • 深度学习

      其实质是构建具有多隐层的感知器神经元网络模型，从低层特征逐步抽象到高层属性，通过对大量样本数据的学习，从而最终提升分类或预测的准确性。

      在遥感中常应用，主要是深度学习的卷积神经网络模型（CNN）。

  • 非监督分类器

    • K-均值算法

    • ISODATA算法

      迭代自组织数据分析算法是在k-均值算法的基础上，增加对聚类结果的“合并”和“分裂”两个操作，并设定算法运行控制参数的一种聚类算法。

  • 决策树分类

    通过学习目标地物与相关要素的分布规律，构建一套基于相关要素的判断规则，通过若干次中间判别，将多个相关要素变量数据集合逐步分解为几个属性均质的特征子集。

    基本思想：逐步从原始数据中分离并掩膜每一种目标作为一个图层或树枝，避免此目标对其他目标提取时造成干扰和影响，最终将所有图层复合以实现对图像的自动分类。

    决策树是一个树型结构，它由一个根结点、一系列内部结点及叶结点组成，每一结点只有一个父结点和两个或多个子结点，结点间通过分支相连。构造决策树是采用自上而下的递归构造方法。

  • 面向对象分类

    基本思想：先将图像分割成多个具有相同属性特征的均质单元（即对象），然后再采用一定的分类算法对这些均质单元进行分类。

    特点：面向对象分类方法分析的基本单元不再是单个像元，而是一个个的均质单元，它充分利用了图像的光谱统计特征以及地物的形状、大小、纹理、相邻关系等一系列信息。

    主要环节：

    • 对象提取（常采用多尺度分割技术）

      • 多尺度分割提取对象

        基本思想：从像元开始由下至上，经过多次迭代运算，将单个像元聚集在多层次更大尺度的分段上，也就是将小的同质区域变成大的同质区域，逐级区域合并的过程。

# 9. 分类后处理

受图像噪声和分类算法本身的影响，传统逐像元的分类结果往往会包含一些由少数几个像元组成的破碎小图斑，它们既可能不符合实际情况，也不太符合视觉习惯。因此在实际应用中，需对这些小斑块进行剔除或者合并到邻近的其它类别。

  • 主/次要分析

    主要分析：将小图斑归并到周边像元数占多数的类别中。这分分类后处理的主要方法

    次要分析：与主要分析相反，它是将像元数占多数的周边类别归并到小图斑中。图像分类结果一般不采用这种方法。

  • 聚类处理

    用形态学膨胀和腐蚀操作将相邻的近似分类区域合并成同一类型，它可以解决分类图像中存在的空间不连续问题。

  • 过滤处理

    将分类结果中较小的类别斑块定义成未分类别（即不考虑小图斑类别的归属问题），也就是把这些小图斑剔除。

    该方法主要是针对分类图像中出现的孤岛问题,过滤处理一般都是在聚类处理之后的图像上进行处理。

  • 其它分类后处理方法

    分类合并：非监督分类预设的分类类别数一般要多于最终所需的类别数，因此在非监督分类结束后，我们还需要根据实际情况将那些具有类似特征的类别进行合并。

    人工修正：对分类结果进行多方面的反复检查，一旦发现有分类错误的地方，如果没法进行自动处理，需进行手工修改。

    矢量数据平滑处理：遥感分类结果是栅格数据，由于其边缘多呈锯齿状，为了满足制图表达的美观要求或其它的矢量处理需求，有时还需要将其转换成矢量数据，并对转换后的矢量边界进行平滑处理。

# 10. 精度评价

验证分类方法的优劣及分类结果的可靠性，为图像分析者对分类方法的改进和用户对分类结果的可信度提供参数支持。

基本思路：在理想条件下，以一幅准确的图像为参考，逐像元对比分类结果与参考图像之间的一致性。现实中不可能存在准确的参考图像来让我们进行逐像元对比分析，因此目前常用的方式是从分类图像中选取一部分检验样本点来进行精度评价。

关键环节：检验样本点选取

常用方法：混淆矩阵、ROC曲线

  • 检验样本的选取

    来源：野外实地调查或基于高分辨率遥感影像解译

    要求：具有代表性，且可以有效的估计总体参数，更重要的是检验样本点要便于野外实地调查或基于高分辨率遥感影像获取其属性类别

    采样方法：简单随机采样、分层采样、集群采样

    个数要求：每个类别的检验样本数量至少得有50个；当区域较大或者分类类别较多时，每类检验样本数量应该有75～100个；样本数量也可根据每个类别的面积比例进行调整。

  • 混淆矩阵

    混淆矩阵也叫误差矩阵，它是通过将每个地表真实像元的位置和类别与分类图象中相应的位置和类别进行比较来评价分类精度。

    • 总体分类精度

      计算：指被正确分类的样本总数除以总样本数。被正确分类的样本沿着混淆矩阵的对角线分布，它显示出被分类到正确的地表真实分类中的样本数。像元总数等于所有地表真实分类中的样本总和。

      作用：表示对每一个随机样本，所分类的结果与地面所对应实际类型相一致的概率。

    • 制图精度

      计算：指样本属于地表真实某类别，分类器能将样本归为该类别的概率。该指标表示地面的某类别被制图者正确分类的概率。

      作用：主要用来评价分类方法的好坏，制图精度越高，说明该分类方法对于该类别分类效果越好。

    • 漏分误差

      计算：指样本属于地表真实某类别，但被分类器分到其他类别中的概率，漏分误差与制图精度是一个互补关系。

      作用：该指标表示地面某类别被制图者分到其他类别的概率，漏分误差越大，则说明该分类方法对于该类别的分类效果越差，有大量该类别的区域未被正确划分，所以在重新进行分类时，应考虑设法把未正确的分类区域划分到该类别中。

    • 用户精度

      计算：指假定分类器将样本归到某类时，相应的地表真实类别是该类别的概率。

      作用：该指标表示分类结果中某类别被正确分类的概率，主要用来评价分类结果的可信度，用户精度越高，则说明该分类结果的可信度越高。

    • 错分误差

      计算：指样本被分为用户感兴趣的类，而实际上属于其他类的概率，错分误差与用户精度是一个互补关系。

      作用：该指标表示分类结果中某类别实际上属于其他类别的概率，错分误差越大，则说明该类别中有大量面积属于其他类别，所以在重新分类时，应设法剔除被错分的区域。

    • Kappa系数

      采样样本发生变化，分类结果的总体精度、制图精度和用户精度可能都会产生较大变化，甚至可能出现无法满足要求的情况。

      Kappa系数采用另一种离散多元统计技术，它综合运用了混淆矩阵的所有参数来计算一个最终指标，从而克服了上述三个精度指标过分依赖于所选样本点的问题。

      判断依据：当Kappa系数处于0.60~0.80时，分类精度较好，当Kappa系数处于0.80~1.00时，分类精度非常好。

  • ROC曲线

    受试者工作特征曲线 (Receiver Operating Characteristic Curve，ROC Curve)，它是一种利用图形来表示分类精度的方法。

    ROC曲线是针对二元分类模型的评价方法。

    • 真正类率(True Positive Rate，TPR)，表示是分类器所识别出的正实例占所有正实例的比例，计算公式为TPR=TP/ (TP + FN)；

    • 假正类率(False Positive Rate，FPR)，分类器错认为正类的负实例占所有负实例的比例，计算公式为FPR= FP / (FP + TN) 。

    ROC曲线精度评价的基本思路：

    在对遥感图像进行二分类时，当给定一个阈值A时将遥感图像分为两类，正类样本将被划分为真正类和假正类，而对于负类样本来说，也会被划分为假负类和真负类，随之产生相对应的真正类率TPR和假正类率FPR 。不同的阈值将会产生与之相对应的TPR和FPR，ROC曲线就是将一系列阈值对应的TPR和FPR的坐标点连接成线。

    ROC曲线评价标准：

    曲线下的面积（Area under the Curve of ROC，AUC of ROC）

    依据：曲线下面积越大，说明该分类方法越可靠。

# 遥感制图表达

遥感制图表达是将遥感数据中的部分或全部信息，按照一定的比例和规则在平面上展现出来。

遥感制图得到的图件具有信息负载、信息传输和图像数据再现与认知的功能，它不仅是研究成果的一种表现方式，也是分析研究的一种重要手段。

遥感制图实际上就是一个从遥感图像中识别、提取信息再表达出来的过程，它可以看作是从遥感图像到制图结果的“翻译”。

# 遥感制图的基本要求

  • 信”：制图表达的信息是可靠的

    要求制图表达的内容是基于原始影像进行合理的分析得到，且具有严密的科学性。

    制图表达最基本、最核心的内容

    两方面要求：

    1. 要求原始数据要准确
    2. 要求图像处理过程要科学

  • “达”：制图表达的信息是清楚且可用的

    需要考虑众多因素（要素内容是否得当？专题图颜色表定义是否符合要求？图像布局中图层的叠放次序是否合适？专题图分类是否符合相关学科的分类标准？……）

    需要较丰富的专业知识和制图经验

    两方面要求：

    1. 要求遥感制图表达过程中需简化要素、提取有用的信息，并选择合适的方式表达出来
    2. 要求制图结果具有实用性

  • “雅”：制图的结果是美观的

    地图的科学内容是通过其特殊艺术形式表达出来。

    考虑众多审美因素（颜色是否搭配？字体是否美观？是否添加了恰当的阴影？文字是否对齐？制图结果布局是否平衡？……）

    具体而繁琐，因人而异

    两方面要求：

    1. 要求制图者技术上过关
    2. 要求制图者在审美上有一定认识

对制图表达的三个要求是依次递进的:“信”是对制图数据源最基本的要求，“达”要求制图者对制图有一定了解，可以准确提取和表达信息，“雅”是出于美观和效果的要求，通常只要做到前两点就可以说是一个合格的制图结果。

# 制图规划

  • 图幅大小

    在制图开始前按照实际情况设置好图幅大小，有利于后续要素添加和布局的设置。

    设定方式：

    • 直接设定图幅大小
    • 根据比例尺设定图幅大小

  • 要素选择

    内容要素：制图表达的主体，包括栅格和矢量数据。

    • 图像空间范围的选择
    • 图像分辨率的选择

    数学要素

    • 比例尺：表示图上距离相较于实际距离的缩放程度
    • 地理信息要素：表示地图方向和地理位置的要素

    辅助要素：图名、图例、文字说明等。

  • 布局

    将制图要素进行规划，将要素安排在制图结果中的适当位置。

    主要考虑要素：

    • 各要素相互之间的位置关系
    • 如果要素在位置上有重合则需注意要素的叠放次序

  • 颜色设计

    • 遥感图像中的波段组合

      灰度图像（单波段）

      真彩色：自然彩色合成图像，图像与实际色彩一致，适合于非专业人士使用。

      假彩色：通过不同的波段组合，强调突出某些在真彩色图像中不好识别的信息，适合于专业人士使用。

    • 专题图中的颜色表定义

      颜色表应具有良好的可区分度

      颜色表应符合心理效应

  • 图像整饰

    图像整饰是对图像的图面进行修整和装饰，整饰主要是为了让制图结果变得更加美观，完成遥感制图表达中“雅”的需要。

    主要包括：

    1. 添加要素（如阴影、图例加框等）
    2. 调整各要素的位置（视觉平衡）
