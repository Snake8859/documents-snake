# 遥感对地观测系统

定义：所谓对地观测，是以地球为研究对象，利用卫星、 航天飞机、飞机等平台所携带的传感器（遥感器） ，对人类生存的地球环境及人类活动本身进行的各 种探测活动。

  • 卫星通信
  • 空间导航定位
  • 遥感和地理信息系统

说明：对地观测可以连续不断地、快速地获取地球表面地物的时空信息，观测地球上发生的各种现象及其变化规律，从而指导人们合理地利用和开发资源，有 效地保护和改善环境，积极地防治和抵御各种自然灾害，改善人类生存和生活的环境质量，以达到有 效发展经济和社会可持续发展的目的

# 根据平台高度划分

  • 天基系统：主要以各种卫星，航天飞机为平台的对地观测系统
  • 空基系统：主要以航空飞机为平台的对地观测系统，包括无人机，有人机等
  • 地基系统：以地面搭建，汽车，船，手持设备等为平台的观测系统

# 根据用途划分

  • 军事观测系统：以军事观测、侦查为目的系统，通常 拥有最先进的尖端技术。
  • 公益性系统：覆盖大区域和全球，对地球各圈层进行 全面和不间断观测与监测，服务于人类社会可持续发 展。这类系统以其数据公开和共享为重要特点，如地 球观测系统（EOS）、地球科学计划（ESE）、以及欧 洲的环境卫星（ENVISAT）系统、民用气象卫星等。
  • 商业系统：以商业应用为主要目标的商业化遥感系统， 如美国GeoEye、 WorldView，中国的吉林系列卫星、 高景系列等卫星。

数据获取——数据处理、评价——数据应用

  • 数据获取：通过航天，航空，地面等哦ing他
  • 数据处理：地面接收系统，图像处理系统
  • 数据质量评价：辐射性能，几何性能，光谱性能评价与分析，进行数据质量控制，为应用提供数据保障
  • 产品生成与分发：地面处理接收系统进行产品处理，分发与共享，提供数据，信息服务
  • 数据应用：应用处理，信息提取

# 遥感卫星数据接收和处理

# 产品级别

  • 0级产品：原始数据
  • 1级产品：相对辐射校正
  • 2级产品：系统几何校正
  • 3级产品：几何精校正
  • 4级产品：正射校正

# 系统运行模式

  • 常规模式

    轨道预报——接收计划——录入归档——处理——分发

  • 订单模式

    根据用户所提交的订购需求进行采集计划安排和产品生成

  • 快速反应模式

    对发生重大突发性事件/时效性要求较高的用户

# 数据处理和产品生成的过程

  • 通过卫星管理部门，获取精确的卫星参数；

  • 建立产品生产的任务单，输入所有必要的处理要求；

  • 加载存档数据，读取所需的全部图象（或信号）及 辅助数据；

  • 如用户需要的是0级（或RAW）产品，则数据处理到此为止，否则继续加工到相应的级别；

  • 对于SAR数据，首先要进行比较复杂成像处理，然后再根据不同的需求分别进行几何变换及地图投影处理等；

  • 对于光学数据，首先进行辐射校正处理，如用户要求1级产品，则数据处理到此为止；如用户 要求更高级别产品，则继续进行相应的几何校正 处理；

  • 将数据处理所得到的结果按照要求的格式生成用户产品；

  • 进行数据产品的检验，并生成规范的产品标志等；

  • 产品经用户服务部交至用户手中。

# 遥感应用的基本步骤

  1. 确定研究方法和技术路线

    进行国内外研究现状调研，确定开展研究工作的总体方案，技术路线和具体方法

  2. 建立分类系统

    根据研究区的情况建立分类图各类及亚类分类指标 （定性、定量），要符合国内外相关标准、规范， 使之具有可比性。

  3. 根据研究 的目标选择合适的遥感数据源

    考虑空间分辨率、时间分辨率、光谱波段、数据类型等因素；

    目标不同、尺度不同、时相要求不同、 光谱特点不同、数据类型与特点不同，需要全面考虑。

    空间分辨率：高空间分辨率用于城市规划、精准农业等；中低空间分辨率用于地质、地貌研究、区域尺度、 全球尺度研究。

    时间分辨率：高时间分辨率用于变化快的目标监测与分析；合适的时相利于目标的识别。

    光谱波段：选择有助于识别目标的波段

    数据种类：可见/近红外、短波红外、热红外、微波数据、激光雷达数据各有不同的特点，根据应用需求 进行选择。

  4. 进行图像的预处理

    根据研究工作的需求，进行多是相图像配准，几何纠正，图像镶嵌，数据融合等。

    图像配准和几何纠正关系？

    区别：

    • 配准：

      • 若在同一地区，同一传感器，不同时相数据，同名点（同一地物）在空间上准确匹配；多景数据；不能保证位置精确
      • 若在同一地区，不同传感器，同名点在空间上准确匹配；多景数据

      配准参考：

      • 地形图内寻找地面控制点GCP
      • 手动采集控制点
      • 找已经几何校正的遥感影像进行配准

    • 几何纠正：消除或减少几何误差几何误差；单景数据；保证位置精确

      起因： （由于平台自身原因，影像本身位置不太精确）

    联系：

    • 在利用已经几何校正的遥感影像进行配准时，同时也进行了几何校正
    • 配准和几何纠正在数学本质上是一致的

  5. 特征参数选择

    为什么要进行特征参数选择？

    1. 减少数据量，加快运算速度和效率
    2. 参与分类的波段越多，分类时需要选择的样本相应地也要增加，否则构成的分类器精度会因为样本不够分反而精度下降，有的类别的样本不是很好选择
    3. 数据冗余影响分类精度，波段的相关性强，影响分类精度，要去冗余，去相关或者减少相关性

  6. 专题信息提取（分类）与综合分析

  7. 结果检验与成果输出
