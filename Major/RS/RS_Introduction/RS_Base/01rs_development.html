# 遥感科学的发展

# 遥感起源与发展

  • 早期：气球，鸽子，空中摄影。

  • 中期:战争推动遥感技术的发展，特别是红外，雷达遥感等。

  • 目前:世界各国向空间发射各类型卫星来对地观测。

  • 航空对地观测和卫星对地观测

历史：

  • 1962年提出遥感（remote sensing）

# 遥感基本概念

# 遥感的定义

  • 广义：即遥远的感知，泛指一切不接触物体而进行的远距离探测，包括对电磁波，力场，机械波（声波，地震波）

  • 狭义：指不与探测目标想接触，利用传感器，把目标的电磁波特征记录下来，处理和综合分析，揭示地物的特性和发展规律。

    【不仅仅是记录，表达，还要对数据进行深度分析】

其他理解：以电磁波与地球表面物质相互作用为基础，来弹出，分析和研究地球资源与环境，并揭示相关要素的空间分布特征与时空变化规律的一门学科。

# 遥感特点

  • 数据覆盖范围大：弥补传统的”点状“信息获取的缺点
  • 时效性强和连续性：时间分辨率和空间分辨率高
  • 多维信息：平面，高层立体数据；多源数据（多平台，多传感器，多光谱，高光谱，多角度）
  • 可视化效果强：以直观图像表达
  • 经济实惠

# 电磁波谱

将电磁波按（或频率）波长从小到大（或逆序）排列起来。

最终形成可见光/近红外，热红外，微波遥感等不同的遥感领域。

# 遥感的基本物理基础

任何物体都具有发射、反射和吸收电磁波的性质， 物体与电磁波的相互作用，形成了物体的电磁波特性，这是遥感探测物体的依据。

# 地物反射波谱曲线

不同类型的地物具有不同的光谱特性，地物电磁波谱特性研究是遥感识别地物的基础。

注意：实验室，地面测量的反射率曲线和卫星实际获得的曲线不是完全一样的。

# 遥感过程

能够对遥感过程进行一定描述。

# 波段

一定的电磁波范围（如TM波段1的波段范围是0.45−0.52μm0.45-0.52\mu m0.45−0.52μm，波段2的范围是0.53−0.61μm0.53-0.61\mu m0.53−0.61μm等）

波段与通道是一致嘛？

可以一致也可以不一样。取决于如何看待。

例如MODIS具有36个波段，也可以说具有36个通道。

在图像合成时，对波段与通道的理解是不一致的。

黑白图像与全色图像

黑白图像和全色波段都是单波段（通道）图像。

黑白中每个灰度值都是由0和1组成。

全色是指全部可见光波段0.38~0.76um，全色图像为这一波段范围的混合图像，空间分辨率更高。

# 彩色合成

多波段合成图像时，将在红、绿、蓝波段获取的单波段图像分别赋予红色、绿色和蓝色，并将3个图像叠加到一起，形成彩色图像。

# 假彩色合成

多波段合成图像时，所用的波段的光谱段和赋予的相应的通道不相一致，合成出来的图像的颜色与实际地物的颜色不相一致。

在彩色合成中，若将TM4，3，2 对应R，G，B，则称为标准假彩色合成。

# 伪彩色合成

单波段图像，只是每个灰度值对应于颜色空间模型中的某一种颜色。

# 分辨率

  • 空间分辨率：指像元所代表的地面范围的大小，即能分辨地面物体的最小单元。

    • 空间分辨率高，划分地物越细，识别地物细节能力强
    • 不一定空间分辨率越高越好，地物细节信息太多有的时候会影响主要信息的识别与提取，因此要根据应用的特点目标选择合适的空间分辨率。
    • 图像的可分辨程度，不完全决定于空间分辨率的具体值，而与目标的形状，大小以及它与周围物体亮度，结构的相对差异有关

  • 光谱分辨率：指传感器接收目标辐射的波谱时能分辨的最小波长间隔。

    其含义通常应该包括：

    • 波长间隔的大小
    • 遥感器波段数量的多少
    • 各波段的中心波长位置

    对应用的意义：光谱信息丰富，可探测到地物光谱的细微变化，微小差异

    高光谱和多光谱数据的区别？

    缺点：光谱分辨高会导致空间分辨率低

  • 时间分辨率：指对同一地点进行遥感数据获取的时间间隔，即重放周期。

    时间分辨率的大小，除了主要决定于飞行器的回归周期外，还与遥感探测器，遥感系统的设计等因素直接相关（如：卫星测摆）

    对应用的意义：动态监测；时序分析

  • 辐射分辨率：是指传感器接收波谱信息时能分布的最小辐射度差（反映探测器的灵敏度，可以通过数据的量化级体现）

# 遥感分类

# 按遥感平台分

  • 地面遥感
  • 航空遥感
  • 航天遥感
  • 宇宙遥感

地面遥感，航空遥感，航天遥感的作用和关系是什么？

地面遥感：

  1. validation（验证）；验证模型，结果等。
  2. calibration（定标）：L=a×DN+biL=a\times DN+biL=a×DN+bi，为获得定标参数
  3. get data（获取数据）
  4. true value （地面真值）
  5. 以点带面

航空遥感：

  1. get data（获得数据）

  2. calibration（定标）

  3. validation（验证）：飞行测验，载荷性能

    特点：

    • flexibility（灵活）：应急能力强

    • 天气影像

    • 空间分辨率高

    • 数据处理难

航天遥感：

  1. 数据重要来源

  2. 卫星多传感器同步

  3. 经济实惠

  4. 处理相对容易

    特点：

    • 空间分辨率低，时间分辨率高

# 按传感器的探测波段分

  • 紫外遥感：0.05~0.38μm\mu mμm（信号比较弱，目前利用还比较少）
  • 可见光遥感：0.38~0.76μm\mu mμm
  • 红外遥感：0.76~1000μm\mu mμm
    • 近红外
    • 短波红外
    • 中红外
    • 热红外
    • 远红外
  • 微波遥感：1mm~1m
  • 多（高）光谱遥感：将波谱划分不同的窄波段进行探测

# 按工作方式分

  • 主动遥感：传感器主动发射电磁波并接收目标的反射回波信号（如SAR【发射微波】，激光雷达LIDAR【发射可见光】）

  • 被动遥感：传感器自身不发射电磁波，被动接收目标对自然辐射源的发射能力或其自身发射的能力。

  • 成像遥感：传感器接收的电磁辐射信号可以转换成图像

  • 非成像遥感：传感器接收的电磁辐射信号不能转换成图像（如微波辐射计，高度计）

    热红外遥感是主动还是被动遥感？

    地物吸收太阳光，发射太阳光，一般属于被动遥感。

    若人为主动发射能力，得到地物能力，又属于主动遥感。

    但星载和机载一般都是被动遥感。

# 遥感的发展趋势

我国的遥感发展能力在不断增强，体现在：

  • 高分专项的实施，GF-1、GF-2、GF-4、GF-3、 GF-5和 GF-6卫星的发射使我国遥感应用能力大为增强
  • 无人机的发展和无人机遥感的普遍应用使遥感出现了大众 创业、万众创新的局面
  • 关于吉林一号
  • 关于高景一号

发展趋势：

  • 多层次：地面，航空，航天
  • 单一传感器——多传感器
  • 分辨率不断提高：空间，时间，辐射和光谱分辨率不断提高
  • 全天候，全天时，全谱段：可见光/近红外，短波红外，热红外，微波
  • 静态——动态：多时相，短周期
  • 定性——定量：新的算法，半自动化，自动化，智能化
  • 遥感平台小型化：灵活，研制周期短，价格便宜
  • 多源数据综合应用：遥感和非遥感资料的有机结合
  • 遥感于GIS，GNSS结合

遥感技术新特点体现在：

  • 三高：高空间分辨率，高光谱分辨率，高时间分辨率
  • 两全：全频段；全方位立体（航天，航空，地面，多角度）
  • 一体化：应用驱动，载荷研制，数据评价与分析，遥感应用

遥感大数据化：

  • Volume（体量大）
  • Variety（种类繁多）
  • Velocity（动态多变）
  • Veracity（冗余模糊）
  • Value（高内在价值）
