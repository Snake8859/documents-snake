# 元胞自动机

# 元胞自动机的基本理论

定义：元胞自动机是一种由细小元胞构成的空间和时间都离散的动力系统，它利用微观的规则、时空离散的状态来模拟宏观的规律和特征。

特点：

  • 元胞自动机是微观的，局部的模型
  • 元胞自动机是离散的动力学模型
  • 元胞自动机运行是基于微观的碰撞规则
  • 同质性、齐性：同质性反映元胞空间内每个元胞变化都服从相同的规律；而齐性是指元胞的分布方式相同、大小、形状相同
  • 离散的（时间、空间、状态）动力学模型
  • 同步计算（并行性）
  • 时空局部性

分类：元胞自动机可以按维数和元胞的类型分类，也可按动力学行为进行分类

  • 平稳型
  • 周期型
  • 混沌型
  • 复杂型

也可以按元胞的形状：

  • 正方形
  • 正六边形
  • 正三角形

满足2NN−2\frac{2N}{N-2}​N−2​​2N​​，其中N必须要整数，解也要整理，则满足的有N=3,4,6

# 元胞自动机的几种模型

  • 生命游戏
  • 交通流模型
  • 林火模型

思路：每一个元胞状态之于前后状态的有关系

# 元胞自动机在地理学中的应用

  • 火山爆发
  • 构造运动
  • 变质作用
  • 对城市交通流的模拟
  • 森林火灾的模拟
  • 城市土地利用的模拟
  • 城市增长的模拟

# 注意点

  • 简单性和真实性的矛盾问题
  • 时空划分问题
  • 微观规则和宏观规则的转换问题
