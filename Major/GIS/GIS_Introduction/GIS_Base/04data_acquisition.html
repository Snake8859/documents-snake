# 元数据

描述数据的数据，描述了数据的内容，质量，条件和其他特征。

用处

  • 可以检索访问数据库
  • 有效利用数据资源
  • 对数据进行加工和二次开发

相当于用户所需要数据的说明书。

分类

  • 按元数据描述对象分类
    • 数据层元数据如日期、投影、位置、误差等；
    • 属性元数据 如数据字典、数据处理规则等
    • 实体元数据如采集原则、有效期等。
  • 按元数据的功能分类

内容

  • 对数据集的描述
  • 对数据质量的描述
  • 对数据处理方法的描述
  • 对数据更新或者集成的说明

# 元数据标准

  • CSDGM（空间元数据内容标准）

    来自USGS，影响大，成为ISO标准

# 空间数据库的数据仓库

数据仓库通常为一个分布式数据库系列，也称作地学数据网关（Geospatial Gateway）

系统是由一系列异地服务器组成，呈树状结构。根节点是总网关，每个结点可以是空间元数据服务器，也可以是一个子网关。

系统要求：

  • 所有空间数据库使用同一元数据标准，如CSDGM
  • 所有网关与服务器之间使用同一协议，如Z39.50，CSW
  • 常用服务软件，如：Isite，GeoNetwork
