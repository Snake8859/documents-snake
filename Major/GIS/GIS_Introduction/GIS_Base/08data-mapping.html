# 地图符号

地图符号是地图的语言，是在地图上表达空间对象 的图形记号，它通过尺寸、形状和颜色来表示事物空间的位置、形状、质量和数量特征，是表达地理 现象与发展的基本手段。

地图符号系统（GIS符号库）反映了表达对象之间 的层次关系，进行了分类分级和抽象的地图符号的 集合。高质量的地图符号丰富了地图的内容、增加 了地图的可读性。

对应于arcgis图层的Properties->Symbology

  1. 符号的位置
  2. 符号的形态
  3. 符号与地图的联动

  • 点状符号

  • 面状符号

  • 点注释

地图格网

ARCGIS中，Data Frame的properties可设置地图格网，包括： Graticule(地理坐标网), Measure Grid, Reference Grid。

图廓整饰中格网的内容主要包括添加经纬网，标注和短线。精心组织图面的布局，多套格网可以并存，如：经纬度不显示格网但在图廓标注、公里格网显示覆盖地图区域的格网线。

# 数据分类

ARCGIS制图过程中，通常要对数据进行分层或分类，然后不同类别的数据进行配色方案、符号配置等制图处理。

数据分类方式：

  • 等间隔：N组数据的数据变化范围或数值间距相等。

  • 几何间隔：几何间隔使得每组元素的平方和最小化。

  • 等频率：即分位数quantile，将数据分为N组且每组内的样本 数相等。

  • 标准离差：原始值减去均值，再除以??。类似于归一化操作。将平均值向上或向下偏移标准值（0.5,1.0等）的单元为类别（或组）的分类点。

    值域空间转换，再另一个值域空间进行分类

  • 自然断点：每一类之间的方差最大，每一类之内的方差最小

配色方案：

  • 单色相方案

    采用同一色相、不同明度和彩度的组合，生成一个系列的色彩方案，如浅红到深红。可以用于显示定量的数据。

  • 双端色方向

    采用两种主色的渐变色。如：有深蓝到浅蓝，再由浅红到深红。适合于有正负值的定量地图获取信息。

  • 光谱方案

    用可见光光谱中全部或部分相邻的色彩来表示显著差异，如：黄色-橙色-红色。全色光谱常用来表示高程地。
