# 数据源

# DEM

数字高程模型(DEM，Digital Elevation Model)是呈规则排列的高程点，通常为规则的格网。来源于摄影测量、遥感LIDAR、散点或等高线插值等。也有使用不规则三角网和等值线模型表达。

# 规则格网

数据模型：规则格网模型

规则格网模型将空间区域分成规则的等距离单元，每个单元对应一个数值，通常在数学上表示为一个矩阵，在计算机中表现为一个二维数组，每个格网单元或数组元素对应一个高程值。

获取方式：用规则方式采样点数据；把不规则采样点数据内插成规则点数据。而后，以矩阵形式来表示地面形状。它已成为栅格数据结构中DEM的通用形式。

# 等值线模型

等值线是DEM模型的平面表示形式，是地形表示中广泛使用的一种表示方法

等值线图以符号化的模型来表示空间立体地形形态。它由一系列数值相等的点，连成的曲线反映连续递变的面状分布地形特征。

它反映的是连续递变的面状分布特征，如等高线、等温线等。

# 坡度(Slope)，坡向(Aspect)

  • 坡度

    地面的坡度是表示地表面上该点倾斜的程度。

    从数学说，地面的坡度是一个矢量，它既有大小，又有方向，坡度定义为地形表面和水平面之间夹角的正切值。用矢量来描述坡度时，坡度是地面的法向量与Z轴之夹角slope 。

  • 坡向

    地面的坡向就是坡面的朝向，粗略的可分为向南、向北、向东、向西。更细的还可进一步分为南、北、东、西、东南、西南、东北和西北向。

    DEM模型中坡向定义为地形表面法线在水平面上的投影与X轴之间夹角。在地学领域中，通常根据法线在水平面上的投影的位置，将其分成阳坡、阴坡、半阳坡、和半阴坡。

参考ArcGIS的操作说明文档，对坡度坡向计算定义。

大致思想，是考虑中心点的上下，左右方向的坡度坡向的加权求和开方值

# 地形因子

  • Topographic Position Index（TPI）

    TPI is defined as the differencee between the elevation at a cell and the average elevation in a cell that surrounds it within a predetermined radius

    半斤可用采用不同大小，如：从10m-200m，甚至更大，反映多尺度变换

    TPI正值表示一个焦点像元的高程比其他窗口内的高程均值大，凸地形；负值则小，凹地形，接近零，平坦

  • Terrain Roughness Index（TRI）

    TRI is defined as the mean difference between a central pixel and its surrounding cells.

    TRI通常用3×3窗口计算，反映了地形的局部起伏的差异性

  • Topographic Wetness Index（TWI）

    TWI is an indicator to represent the spatial distribution of water flow and water stagnation across a watershed.

    twi=log(accumulation/tan(slope))twi = log(accumulation/tan(slope)) twi=log(accumulation/tan(slope))

    Multiple flow direction algrithms(MFD) is used to calculate a catchment area, instead of single flow direction as did by ARCGIS.

# 流域分析

# 流域的基本概念

流域是指地表水及地下水分水线所包围的集水区域的总称，**习惯上常指地表水的集水区域。**根据地形图上的分水线，可以定出流域边界，此范围内的面积，称为流域面积。

# 河流分级

  • Strahler method

    STRAHLER外连接总被赋值为1。**但当同样级别的水系交叉时，水系级别增加。**所以，两个第二级的水系交叉时，会生成第三级的水系。**但当两个不同级别的水系交叉时，则不会产生水系别的增加，而保持较高级别的水系等级。**例如，第一和第二级的水系交叉不会产生第三级的水系，

  • Shreve method

    Shreve法计算水系网中的所有连接数，外连接被赋值为1，但对内连接其级别数是增加的。例如，两个一级支流交叉时会产生二级支流，而一级支流和二级支流交叉时则生成三级支流。

# 水流方向

水流向定义：水流离开每一个栅格单元时的指向，水流方向的表示方法如下图所示。

一般采用1，2，4，8，16，32，64，128的2的整数次幂来表达栅格单元的8领域方向

当前被处理单元的水流向的确定：重力作用，自高而低；坡度最大原则。

流向判定建立在3×3 的DEM 栅格网的基础上，其方法有单流向法和多流向法之分，但单流向法因其确定简单、应用方便而应用广泛。

常用算法：

  • D8

    计算当前被处理栅格单元与8个相邻的像元高度落差，然后再除以他们之间的欧式距离，选取值最大的地方（即坡度最大）为该像元流出的方向，同时根据之前所定义的水流方向表达，将其单元值表达为1，即水流方向是向右。

  • D8 (commonly used) cannot represent divergent flow (peaks/ridges, flat areas)

    D8算法，不能表达不同的流量

# 流域分析算法流程

DEM填洼处理

  • 洼地或者凹陷（sink）

    A sink is a cell or set of spatially connected cells whose flow direction cannot be assigned one of the eight valid values in a flow direction grid. This can occur when all neighboring cells are higher than the processing cell, or when two cells flow into each other creating a two-cell loop.

    洼地类型：

    1. 错误地表描述，产生原因：数据采集，数据内插
    2. 真实的地表描述，产生原因：湖泊，喀斯特地貌，陷穴

  • 填充流程

    1. 洼地深度计算

      洼地填充前，计算洼地深度判断哪些由数据误差造成，哪些又是真实地表形态，设定阈值。

      洼地深度：洼地区域最低高程值与洼地区域最低出水口高程之差。

    2. 洼地填充

流域划分与水系提取

河网提取的主要方法：地表径流漫流模型

主要思路：

  1. 基于无洼地DEM利用最大坡降法计算水流流向
  2. 依据自然水流由高处流往低处的规律，基于水流流向数据计算汇流累积量；
  3. 假设每个Cell都携带一份水流，那么汇流累积量就代表着该Cell的水流量。当汇流量达到一定值时，就会产生地表径流。挑选所有汇流量大于临界值的栅格就是（潜在的）水流路径，由这些水流路径构成的网络，就是河网。

具体操作如下：

  • 计算汇流

    **原理：**基于栅格间水流流向判别，核心是计算每个栅格点上游汇流区的面积（或者是汇流累积的格网数）即汇流累积量。

    基本思想：以规则格网表示的数字地面高程模型每点处有一个单位的水量，按照水流从高处流往低处的自然规律，根据区域地形的水流方向数据计算每点处所流过的水量数值，从而得到该域汇流累积量 。

  • 河网生成

    基于DEM生成汇流累积量数据，用到洼地填充、水流流向、生成汇流累积量一组方法。

    **设定阈值。**不同级别的沟谷对应不同的阈值，不同区域相同级别的沟谷对应的阈值也是不同。实际工作中设定阈值应通过不断实验、结合土壤植被等资料确定。

    **生成栅格河网。**采用Map Algebra/Multi Map Output工具中的Con或Setnull命令条件查询可得栅格河网。或采用Raster Calculator计算。

    **栅格河网矢量化。**Hydrology/Stream to Feature，生成矢量河网

  • 流域分割

    流域（watershed）又称集水区域，是指流经其中的水流和其他物质从一个公共的出水口排出从而形成的一个集中的排水区域，也可以用流域盆地（basin）、集水盆地（catchment）等来描述。

    Watershed显示了每个流域汇水区域的大小，流域间的分界线即为分水岭。分水线包围的区域称为一条河流或水系的流域。

    主要思路：

    1. 流域盆地确定

      ArcGIS中的hydrology工具的basin工具所提取出的所有流域盆地均是认为盆地的出水口在研究区域的边缘。

    2. 汇水区出水口确定

      可采用Hydrology/Snap pour point，以记录潜在但并不准确的小级别流域出水口位置的点数据为基础，搜索该点一定范围内汇流累积量较高的栅格点（即为小级别流域的出水口）。

    若没有出水点的栅格或矢量数据，可利用已生成的StreamLink作为汇水区的出水点。

    1. 集水流域生成

      先确定一个出水点，也就是该集水区的最低点，然后结合水流方向数据，分析搜索出该出水点上游所有流过该出水口的栅格，直到所有的该集水区的栅格都确定了位置，也就是搜索到流域的边界，分水岭的位置。

  • 小流域和HRU个数的合理划分

    划分方法：

    • 子流域的划分是利用DEM数据根据用户设定的汇流面积阈值。模型会提供一个建议使用阈值。

    • HRU的划分

      • 面积最大的土地利用类型和土壤类型的组合

      • 多个不同土地利用类型和土壤类型的组合

        土壤利用类型和土壤面积的最小阈值比，小于设定的阈值，则在模拟中不予考虑，余下的土地利用和土壤类型的面积将重新按比例计算，以保证整个子流域的面积得到100%
