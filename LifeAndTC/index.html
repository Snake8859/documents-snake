# 生活

# 2020

# 毕业——挥手之间，时过境迁

从2019年末开始，生活节奏就发生微妙的变化，2019年11月份选择提前去北京体验研究生生活，人生第一次离开长沙，远赴他乡求学，在新技术，遇到了新的老师们和朋友们，也算是有了一份新的开始。

2020年1月，回到长沙，和本科玩得最好的小伙伴们一起聚餐，闲聊碎事，讨论毕设，吐槽生活，想象毕业旅行。可没想到一场疫情，打破这一切安排，大家各自家乡度过剩下的日子。大四了，没有课，只剩下毕设，大家只能在互联网上相互交流，大家之间联系依靠无数根网线，依靠着各种APP......

这样日子，一直到5月中旬，返校的消息有了眉头之后，大家都返回了陌生而又熟悉的校园，表面像往常一样生活，内心却依然有了毕业的忧愁。那段日子，各种聚餐，聚会，然后是答辩，以及最后的毕业照。相对于大多数其他学校的同学来说，我们是幸运的，还能和同学们聚餐，还有毕业照，还能再一次相会。

大学四年不快也不慢，该做的事都做了，遗憾的事也遗憾了，酸甜苦辣尽在其中。这四年谈不上有多么多么努力，多么多么勤奋，大多数都是秉承着不想留下任何遗憾的心态，用自己喜欢的方式向前走。目前的这一切，我觉得这只是在这样的心态下的各种巧合，但不保证未来时时刻刻都有这样巧合，希望在毕业之后，能将这样不确定性巧合转为确定性选择。

# 毕业啦，挥手之间，时过境迁，我们未来江湖相见！

# 2021

# 研一秋季

2020年9月迎来了国科大研一秋季的开学，本以为在北京读书肯定体验到北京大城市的繁华，但没想到是在怀柔区，北京郊区的郊区。国科大虽然地区不繁华，但是科研还是很繁华，在这里遇到来自各个学校的优秀的同学们，能来到这里认识大家，就值得了。

# 食堂

食堂价位和味道一般，不过学校和科研所每个月补助已经基本上足够在食堂消费了，一月感觉花800-900左右。

# 宿舍

宿舍，我还是很满意的，以前觉得单人间挺好，可现在想想其实也没那么好，单人间有时候自己一个人孤独就算了，有时候一个人还容易忘事情。现在我们是双人间，有的同学是三人间，相互之间有照料，不会那么无聊，室友和我都是一个性格的人，感觉挺好的，整个套间也相处十分融洽。大客厅让人有种公寓的氛围，卫生间和浴室也单独分开，大家洗澡之类的时间都是足够的。

# 教室

和大学的感觉是一样的，没啥区别，非要说的话，有暖气，在室内容易中暑，哈哈哈。

# 教学

不得不说，自选课程是一个亮点，不过大多数时候都会考虑和自己专业，研究相关的课程。国科大的课都是十分优秀的，对于我来说有些内容学起来还是困难的，所以我比较适可而止，没用选太多太难的课。这学期比较困难的课，主要是有图像处理和模式识别导论，专业课地信和遥感中规中矩，还有英语课。选课是一个难事，选的好和选的不好都会不同程度影响，所以这个要慎重。
